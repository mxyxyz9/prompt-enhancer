## Description
AI-powered Chrome extension that enhances user prompts for better AI responses using Google's Gemini API.

## Features
- One-click prompt enhancement
- Secure API key storage
- Cross-site compatibility
- Error handling and notifications

## Installation
1. Clone repository:
```bash
git clone https://github.com/yourusername/prompt-enhancer.git
```
2. Install dependencies:
```bash
npm install
```
3. Build extension:
```bash
npm run build
```

## Usage
1. [Get Google Gemini API Key](https://ai.google.com)
2. Right-click extension icon → Options
3. Enter API Key and click Save
4. Visit any AI chat interface and look for the "Enhance" button

## Configuration
- Requires Node.js v18+
- API key stored securely using Chrome's sync storage

## Troubleshooting
❗ "API key not set" error:
- Reload extension
- Verify key in Options
- Ensure key has Gemini API access
